# Nadav Weisler

I build experimental software, hackathon projects, and side projects around psychology, research, and whatever seems interesting enough to make.

Some of my projects are built for actual research use. Others started as experiments, jokes, or weekend ideas and somehow became real.

## Things I like building

- tools for psychology and behavioral experiments
- jsPsych / PsychoPy-based experiment components
- hackathon projects with real-world value
- small web apps and side projects
- automation and data tools when manual work gets too annoying
